A clean and simple weather forecast app built with HTML and CSS.

---

## Features

- Displays current weather conditions for Tokyo  
- Shows a 3-day forecast with highs and lows  
- Interactive hover effects on forecast cards

---

## Technologies

- HTML5  
- CSS3
